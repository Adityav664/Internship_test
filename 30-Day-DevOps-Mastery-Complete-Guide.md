## Week 1: Linux Fundamentals, Bash Scripting, Networking
### Day 1: Linux Basics
**Learning Objectives:**
- Understand the Linux file system
- Learn basic command line commands

**Theory:**
Linux is an open-source operating system that offers high flexibility and security. There are many flavors of Linux, such as Ubuntu, CentOS, and Debian. 

### Practical Commands:
```bash
# To check the current directory
pwd

# To list files
ls -la

# To create a directory
mkdir new_directory
```

### Day 2: Intermediate Linux
**Learning Objectives:**
- Learn file permissions and ownership
- Understand processes and jobs

**Theory:**
Linux file permissions control who can read, write, or execute files. Processes can be running in the foreground or background.

### Practical Commands:
```bash
# To change directory permissions
chmod 755 myfile

# To view running processes
top
```

### Practice Exercises:
